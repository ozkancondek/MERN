# MERN-Bookstore Project

### MongoDB Express.js React Node.js

- Project is created to better understanding of frontend and backend interaction (MongoDB, Express.js, React, Node.js)
- Mostly worked on backend side to practice, router, controllers, mongoose, token logic and middleware
- Ant Design is used to style client side

---

## Tech Stack

1. Backend, node.js server(Restful Api)

2. Database MongoDB

3. Frontend, React

4. Payment(Stripe)

5. Production, Heroku
